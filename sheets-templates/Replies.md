# Replies Sheet Template

## Sheet Name: "Replies"

## Column Structure

| Column | Header | Data Type | Description | Formula/Notes |
|--------|--------|-----------|-------------|---------------|
| A | Property ID | Text | Links to Properties sheet | Foreign key reference |
| B | Email Thread ID | Text | Gmail thread ID | Links to Emails sheet |
| C | Reply Timestamp | DateTime | When reply was received | Auto-populated by Apps Script |
| D | Sentiment | Text | AI-classified sentiment | "Interested", "Counteroffer", "Not Interested", "Spam", "Neutral" |
| E | Reply Type | Text | Type of response | "Acceptance", "Counteroffer", "Rejection", "Question", "Unclear" |
| F | AI Confidence | Number | AI confidence score (0-100) | Percentage format |
| G | Original Offer | Currency | Original offer amount | Lookup from Properties or Emails |
| H | Counteroffer Amount | Currency | Counteroffer if present | Extracted by AI |
| I | Notes | Text | AI-generated summary | Full response summary |
| J | Next Action | Text | Recommended next step | Auto-generated by AI |
| K | Action Taken | Checkbox | Whether action was executed | TRUE/FALSE |
| L | Handler Status | Text | Workflow status | "Pending", "Processed", "Follow-up Sent", "Marked Dormant" |

## Data Validation Rules

- **Sentiment Column (D)**: Dropdown: "Interested", "Counteroffer", "Not Interested", "Spam", "Neutral"
- **Reply Type Column (E)**: Dropdown: "Acceptance", "Counteroffer", "Rejection", "Question", "Unclear"
- **Handler Status (L)**: Dropdown: "Pending", "Processed", "Follow-up Sent", "Marked Dormant"

## Conditional Formatting

1. **Interested**: Sentiment = "Interested" → Green background
2. **Counteroffer**: Sentiment = "Counteroffer" → Yellow background
3. **Not Interested**: Sentiment = "Not Interested" → Red background
4. **Spam**: Sentiment = "Spam" → Gray background
5. **High Confidence (>80%)**: F column → Green text
6. **Low Confidence (<50%)**: F column → Red text

## Lookup Functions

- **Property Address**: `=IFERROR(VLOOKUP(A2, Properties!A:B, 2, FALSE), "")`
- **Agent Name**: `=IFERROR(VLOOKUP(B2, Emails!H:C, 2, FALSE), "")`
