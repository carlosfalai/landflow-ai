# Emails Sheet Template

## Sheet Name: "Emails"

## Column Structure

| Column | Header | Data Type | Description | Formula/Notes |
|--------|--------|-----------|-------------|---------------|
| A | Property ID | Text | Links to Properties sheet | Foreign key reference |
| B | Agent Email | Text | Recipient email address | Email format validation |
| C | Agent Name | Text | Recipient name | For personalization |
| D | Subject | Text | Email subject line | Auto-generated by template |
| E | Status | Text | Email status | "Draft", "Sent", "Delivered", "Opened", "Replied" |
| F | Timestamp | DateTime | When email was sent | Auto-populated by Apps Script |
| G | Message ID | Text | Gmail message ID | For thread tracking |
| H | Thread ID | Text | Gmail thread ID | Links related emails |
| I | Follow Up Date | Date | Next follow-up date | `=F2+2` for 48-hour follow-ups |
| J | Follow Up Count | Number | Number of follow-ups sent | Auto-incremented |
| K | Email Type | Text | Type of email | "Initial", "Follow-up", "Counteroffer", "Confirmation" |

## Data Validation Rules

- **Status Column (E)**: Dropdown: "Draft", "Sent", "Delivered", "Opened", "Replied"
- **Email Type Column (K)**: Dropdown: "Initial", "Follow-up", "Counteroffer", "Confirmation"
- **Agent Email (B)**: Data validation for email format

## Conditional Formatting

1. **Sent Emails**: Status = "Sent" → Green background
2. **Replied**: Status = "Replied" → Blue background
3. **Needs Follow-up**: Follow Up Date < TODAY() and Status ≠ "Replied" → Yellow background

## Formulas

- **Follow Up Date (I)**: `=IF(F2<>"", F2+2, "")` (48 hours after send)
- **Days Since Send**: `=IF(F2<>"", TODAY()-F2, "")`

## Lookup Functions

- **Property Address**: `=IFERROR(VLOOKUP(A2, Properties!A:B, 2, FALSE), "")`
- **Property Price**: `=IFERROR(VLOOKUP(A2, Properties!A:D, 4, FALSE), "")`
